# Bootcamp Tascom & ETE JBL - Lista de Exercícios 02

Este repositório foi criado para documentar e organizar a resolução da lista 2 do Bootcamp Tascom realizado em parceria com a ETE JBL.

## Sobre o Projeto
O objetivo desta etapa foi consolidar os pilares fundamentais da programação, garantindo uma base sólida para o desenvolvimento de sistemas mais complexos.

## O que foi aprendido
* Laços de repetição (for, while, do while)
* Estruturas condicionais (if, else, else if)
* Exibição de dados no console.
* Versionamento de código utilizando Git e GitHub.

## Tecnologias utilizadas
* JavaScript
* Node.js
* Git e GitHub


## Como usar
Com o Node.js instalado, siga os passos abaixo:

1. Abra o terminal e clone o repositório utilizando:
```bash
git clone https://github.com/Alissomberg/BootcampTascom-lista3.git
``` 
2. Abra um terminal na pasta dos exercícios e então digite:
```bash 
node nome-do-arquivo.js
